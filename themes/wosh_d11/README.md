# Wosh D11 Theme

## Description

Wosh D11 est un thème Drupal 11 moderne et responsive, basé sur le thème Wosh original mais entièrement adapté pour l'API Drupal 11. Ce thème offre une expérience utilisateur exceptionnelle avec des fonctionnalités avancées et une compatibilité complète avec les dernières versions de Drupal.

## Fonctionnalités

### Compatibilité
- **Drupal 11** : Entièrement compatible avec l'API Drupal 11
- **Drupal 10** : Compatible également avec Drupal 10
- **Responsive Design** : Optimisé pour tous les appareils

### Bibliothèques incluses
- **Bootstrap 5.3.0** : Framework CSS moderne
- **Font Awesome 6.4.0** : Icônes vectorielles
- **Ionicons 7.1.0** : Icônes supplémentaires
- **Owl Carousel 2.3.4** : Carrousel responsive
- **Animate.css 4.1.1** : Animations CSS
- **jQuery plugins** : Diverses extensions jQuery

### Fonctionnalités du thème
- **Système de couleurs** : 9 schémas de couleurs prédéfinis
- **Paramètres de thème avancés** : Interface d'administration complète
- **Layouts flexibles** : Support de plusieurs mises en page
- **Commerce ready** : Optimisé pour Drupal Commerce
- **SEO friendly** : Structure optimisée pour le référencement

## Installation

1. Téléchargez ou clonez le thème dans le dossier `themes/` de votre installation Drupal
2. Activez le thème dans l'administration Drupal : `Appearance > Themes`
3. Configurez les paramètres du thème : `Appearance > Settings > Wosh D11`

## Configuration

### Paramètres généraux
- **Preloader** : Affichage d'un preloader au chargement
- **Scroll to top** : Bouton de retour en haut de page

### Layouts
- **Wide Layout** : Mise en page large
- **Framed Layout** : Mise en page encadrée
- **Boxed Layout** : Mise en page en boîte

### Header
- **Header Style** : Choix entre header par défaut et transparent
- **Menu Position** : Position du menu (centre, gauche, droite)
- **Sticky Header** : Header fixe lors du scroll

### Footer
- **Footer responsive** : Tailles automatiques ou manuelles
- **Footer colors** : Choix de couleurs (clair, sombre, coloré, plain)

### Commerce
- **Product layouts** : 4 layouts différents pour les produits

## Régions disponibles

- **Branding** : Logo et nom du site
- **Primary Menu** : Menu principal
- **User Account** : Menu utilisateur
- **Search** : Recherche
- **Cart** : Panier (Commerce)
- **Header** : En-tête personnalisé
- **Slider** : Zone slider
- **Page Title** : Titre de page
- **Breadcrumb** : Fil d'Ariane
- **Content Wide Top** : Contenu large haut
- **Content** : Contenu principal
- **Sidebar First/Second** : Barres latérales
- **Content Wide** : Contenu large bas
- **Footer** : Zones de footer (4 widgets + 2 bottom)

## Breakpoints

Le thème utilise les breakpoints Bootstrap 5 :
- **XS** : < 576px
- **SM** : 576px - 767px
- **MD** : 768px - 991px
- **LG** : 992px - 1199px
- **XL** : 1200px - 1399px
- **XXL** : > 1400px

## Personnalisation

### CSS personnalisé
Utilisez l'interface d'administration pour ajouter du CSS personnalisé sans modifier les fichiers du thème.

### Couleurs
9 schémas de couleurs sont disponibles :
- Default (Rouge)
- Yellow
- Blue
- Green
- Pink
- Orange
- Orange Red
- Red
- Brown

### Templates
Tous les templates Twig sont disponibles dans le dossier `templates/` et peuvent être surchargés selon vos besoins.
